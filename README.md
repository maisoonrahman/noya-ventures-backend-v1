# Noya Ventures App
This is the backend repo for Noya Ventures

## Overview:
A minimal Flask REST API backend for startups, investors, and partners to connect around the world.

Links:
test: test.noya.vc
prod: noya.vc

Handles:
* User, startup, and founders data (CRUD via API)
* PDF document uploads for startup verification
* JSON responses for frontend consumption (React frontend in a separate repo)

This backend is designed to be modular, simple, and easily connected to a MySQL database.
